Climate and Ecosystem Resilience of Jammu & Kashmir

GIS-Based Spatial Modeling using QGIS

Project Overview

This GIS-based study evaluates ecosystem resilience across Jammu & Kashmir by integrating climate, vegetation, and terrain variables into a unified spatial model. The goal is to derive a Resilience Index (RI) representing environmental stability and vulnerability patterns.

Conceptual Framework

Ecosystem resilience is modeled as interaction between ecological condition, climatic stress, and terrain sensitivity. All variables are normalized to ensure comparability and integrated using a multi-criteria GIS framework.

Data & Variables

  • NDVI (Vegetation health)
  • Temperature (Climate stress)
  • Precipitation (Moisture availability)
  • DEM-derived Slope & Aspect
  • Land Use/Land Cover

Index Formulation

Adaptive Capacity Index (ACI)

ACI = (NDVI_norm + Forest_cover_norm) / 2

Climate Stress Index (CSI)

CSI = (Temperature_norm + Precipitation_norm) / 2

Ecosystem Sensitivity Index (ESI)

ESI = (Slope_norm + Aspect_norm + LULC_norm) / 3

Resilience Index (RI)

RI = (ACI + (1 - CSI) + (1 - ESI)) / 3

Interpretation

The results reveal spatial heterogeneity in ecosystem resilience across Jammu & Kashmir. High resilience zones correspond to stable vegetation and lower climatic stress, while low resilience zones indicate higher environmental vulnerability.
