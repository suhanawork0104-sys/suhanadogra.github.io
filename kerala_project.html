Project Overview

This project develops an Environmental Vulnerability Index (EVI) for Kerala using GIS, remote sensing and Python. Multiple environmental indicators including Digital Elevation Model (DEM), slope, rainfall, vegetation (NDVI), and land cover were integrated into a weighted model to evaluate environmental vulnerability across the state.


The project combines Google Earth Engine for remote sensing, QGIS for spatial processing, and Python for raster analysis, statistical calculations, and development of an interactive Streamlit dashboard. The final output supports environmental assessment through interactive visualization and spatial analysis.

Methodology

The Environmental Vulnerability Index (EVI) was developed by integrating five environmental indicators representing terrain, vegetation, climate and land cover characteristics. Each dataset was preprocessed, normalized, assigned a weight according to its contribution to environmental vulnerability, and combined using a weighted overlay approach.

Environmental Vulnerability Index (EVI)

The Environmental Vulnerability Index (EVI) was developed using a weighted overlay approach by integrating five environmental indicators representing terrain, vegetation, climate, and land cover.


Weighted Overlay Formula

EVI =

(0.15 × DEM) + (0.30 × Slope) + (0.20 × NDVI) + (0.25 × Rainfall) + (0.10 × Land Cover)


Each raster layer was normalized before applying the assigned weights. Higher EVI values indicate greater environmental vulnerability, while lower values represent relatively stable environmental conditions.

Interpretation

The Environmental Vulnerability Index highlights spatial variations in environmental conditions across Kerala. Areas with higher EVI values represent relatively greater environmental vulnerability due to the combined influence of terrain, rainfall, vegetation, and land-cover characteristics, whereas lower values indicate comparatively stable environmental conditions.
