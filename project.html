Climate & Ecosystem Resilience of Jammu & Kashmir

GIS-Based Spatial Modeling using QGIS

Project Overview

This project evaluates ecosystem resilience across Jammu & Kashmir using GIS and remote sensing. It integrates climate, vegetation, and terrain variables into a unified Resilience Index (RI).
