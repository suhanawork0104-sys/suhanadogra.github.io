Climate and Ecosystem Resilience of Jammu & Kashmir

GIS-Based Spatial Modeling using QGIS

Project Overview

This project evaluates ecosystem resilience across Jammu & Kashmir using a GIS-based framework integrating climate, vegetation, and terrain variables. The goal is to generate a Resilience Index (RI) representing spatial ecological stability and vulnerability.
