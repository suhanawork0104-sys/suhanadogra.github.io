Climate & Ecosystem Resilience of Jammu & Kashmir

GIS-Based Spatial Modeling using QGIS

Project Overview

This project evaluates ecosystem resilience across Jammu and Kashmir using GIS and remote sensing. The goal is to generate a unified Resilience Index (RI) using climate, vegetation, and terrain variables.

Conceptual Framework

Ecosystem resilience is modeled as interaction between climate stress, vegetation condition, and terrain sensitivity.

Data & Variables

NDVI • Temperature • Precipitation • DEM (Slope & Aspect) • Land Cover
