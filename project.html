Climate and Ecosystem Resilience of Jammu & Kashmir

GIS-Based Spatial Modeling using QGIS

Project Overview

This project evaluates ecosystem resilience across Jammu and Kashmir using a GIS-based multi-criteria framework integrating climate, vegetation, and terrain variables. The objective is to derive a single Resilience Index (RI).

Conceptual Framework

Ecosystem resilience is modeled as a balance between ecological condition, climatic stress, and terrain sensitivity using normalized geospatial variables.

Data & Variables

  • NDVI (Vegetation health)
  • Temperature (Climate stress)
  • Precipitation (Moisture availability)
  • Slope & Aspect (Terrain sensitivity)
  • Land Use/Land Cover (Human + ecological structure)

Methodology Flow

WorldClim
Temp + Precip
↓ CSI
Landsat NDVI + Forest Cover
↓ ACI
Slope + Aspect + LULC
↓ ESI
ACI + CSI + ESI
↓
Resilience Index (RI)

Index Formulation

Adaptive Capacity Index (ACI)
ACI = (NDVI_norm + Forest) / 2

Climate Stress Index (CSI)
CSI = (Temperature_norm + Precipitation_norm) / 2

Ecosystem Sensitivity Index (ESI)
ESI = (Slope_norm + Aspect_norm + LULC_norm) / 3

Resilience Index (RI)
RI = (ACI + (1 − CSI) + (1 − ESI)) / 3

Final Output Maps

Resilience Index (RI)

Interpretation

Spatial variation in ecosystem resilience reflects interactions between vegetation health, climatic stress, and terrain complexity across Jammu and Kashmir.
